Jekyll Blogging How-to

based on:
https://onextrapixel.com/start-jekyll-blog-github-pages-free/

1. cd /srv/gitblog/marshall62.github.io/voice-coding-blog
2. jekyll serve --watch
3. visit http://127.0.0.1:4000/

Config:
edit _config.yml and about.markdown


4. To create a new entry:
cd _posts folder
create file with name like year-month-date-{slug}.md | .markdown


To Publish:
 - make sure I have a github repo with the same name as the blog (e.g. marshall62/voice-coding-blog
 
 git init
 git add .
